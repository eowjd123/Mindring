# 노션 페이지 ID 찾는 방법

## 📋 페이지 ID 추출 방법

### 방법 1: URL에서 추출

1. 노션에서 개발 내역을 작성할 페이지를 엽니다
2. 브라우저 주소창의 URL을 확인합니다

**예시:**
```
https://www.notion.so/Mindring-Development-Log-abc123def456ghi789jkl012mno345pq
```

3. URL의 마지막 부분이 페이지 ID입니다
   - 페이지 ID: `abc123def456ghi789jkl012mno345pq`
   - **하이픈(-)을 제거**해야 합니다
   - 최종 페이지 ID: `abc123def456ghi789jkl012mno345pq`

### 방법 2: 페이지 링크 복사

1. 노션 페이지에서 우측 상단 "..." 메뉴 클릭
2. "Copy link" 선택
3. 복사된 링크에서 페이지 ID 추출

**예시:**
```
https://www.notion.so/Mindring-Development-Log-abc123def456ghi789jkl012mno345pq?v=...
```

페이지 ID는 `-` 다음의 32자리 문자열입니다 (하이픈 제거).

### 방법 3: 페이지 설정에서 확인

1. 노션 페이지에서 우측 상단 "..." 메뉴 클릭
2. "Settings" 또는 "페이지 설정" 선택
3. 페이지 URL에서 ID 확인

## ⚠️ 중요 사항

1. **Integration 연결 필수**
   - 페이지에서 "..." → "Add connections" 클릭
   - "MINDRING" Integration 선택하여 연결
   - 연결하지 않으면 스크립트가 작동하지 않습니다

2. **페이지 ID 형식**
   - 32자리 영문자와 숫자 조합
   - 하이픈(-)은 제거해야 합니다
   - 예: `abc123def456ghi789jkl012mno345pq`

3. **빈 페이지 권장**
   - 스크립트는 페이지에 내용을 추가합니다
   - 기존 내용이 있으면 그 아래에 추가됩니다
   - 깔끔한 정리를 위해 빈 페이지를 권장합니다

## 🔧 .env 파일에 추가하기

페이지 ID를 찾은 후, **로컬 개발 환경에서만** 사용하는 `.env` 파일에 아래와 같이 환경변수를 추가하세요.

- 노션 API 토큰용 환경변수 (예: `MY_NOTION_SECRET`)
- 노션 페이지 ID용 환경변수 (예: `MY_NOTION_PAGE_ID`)

> ⚠️ 실제 토큰 값과 `.env` 파일 자체는 **절대 Git 저장소에 올리지 마세요.**  
> 이 가이드는 키 이름만 예시로 안내하며, 실제 값은 각자의 로컬 환경에서만 관리해야 합니다.

## ✅ 확인 방법

페이지 ID를 확인했으면 다음 명령어로 테스트하세요:

```bash
npm run notion:sync
```

성공하면 노션 페이지에 개발 내역이 자동으로 작성됩니다!
